Как это работает

Загрузите Excel файл (.xlsx или .xls) и укажите необходимые столбцы:

  • Столбец с телефонами - номера будут приведены к формату 79998887766
  • Столбец с типом участника - для каждого типа будет создан отдельный файл
  • Столбец для сортировки (опционально) - участники будут отсортированы по этому столбцу

Результаты будут представлены в виде нескольких файлов для скачивания.

{% with messages = get_flashed_messages() %} {% if messages %} {% for message in messages %}
{{ message }}
{% endfor %} {% endif %} {% endwith %}
